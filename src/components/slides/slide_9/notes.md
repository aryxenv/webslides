# Revenue lives in repeat support

Use this slide to shift from sports impact to fundraising behavior. Directly selling data is hard and not the strongest angle. The stronger angle is using data to improve sponsorship and donation strategy.

The key idea is retention. Existing supporters are more likely to give again, but new donor retention can be very low. The provided draft references 19.4% of new donors giving again the following year and a cost contrast where acquisition is much more expensive than retention.
